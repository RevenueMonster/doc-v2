**Method :** <span style={{ color: "orange", fontWeight: "bold" }}>POST</span><br/>

URL: `https://api.alacarte.my/open/v1/store/{id}/category`<br/>

Sandbox URL: `https://sb-api.alacarte.my/open/v1/store/{id}/category`

:::note

- Using `storeId` `name` and `status` to create a category

:::

### Request Parameters

| Parameter | Type   | Description                                        | Example  |
| --------- | ------ | -------------------------------------------------- | -------- |
| `name`    | String | Category name                                      | "Drinks" |
| `status`  | String | Status of Category **"ACTIVE"** and **"INACTIVE"** | "ACTIVE" |

| Parameter    | Type     | Description                                        | Example                          |

| `categoryId` | String   | Category ID                                        | "5f697cb3585d7cc1ae35a37f"       |

| `itemIds`    | Array    | Category Items                                     | null                             |
