**Method :** <span style={{ color: "orange", fontWeight: "bold" }}>POST</span><br/>

URL: `https://open.revenuemonster.my/v3/service`<br/>

Sandbox URL: `https://sb-open.revenuemonster.my/v3/service`

:::note

- Update the category **name** , **status** and **positionChange**

:::

### Request Parameters

| Parameter  | Type   | Description                       | Example                      |
| ---------- | ------ | --------------------------------- | ---------------------------- |
| `service`  | String | alacarte service                  | "alacarte"                   |
| `version`  | String | Currently only support 'v1"       | "v1"                         |
| `function` | String | Function that provide by alacarte | "update.category"            |
| `request`  | Object | Refer to explanation below        | (Refer to explanation below) |

| Parameter        | Type   | Description                                                | Example                    |
| ---------------- | ------ | ---------------------------------------------------------- | -------------------------- |
| `categoryId`     | String | ID of the Category                                         | "5f68485ccb3285b9d3955696" |
| `name`           | String | Category name                                              | "Drinks"                   |
| `status`         | String | Status of Category **"ACTIVE"** and **"INACTIVE"**         | "ACTIVE"                   |
| `positionChange` | Int    | Move **UP** or **DOWN** the position using **-1** or **1** | 1                          |
