| `function` | String | Function that provide by alacarte | "get.category.items"         |
| `request`  | Object | Refer to explanation below        | (Refer to explanation below) |

| Parameter    | Type   | Description        | Example                    |
| ------------ | ------ | ------------------ | -------------------------- |
| `categoryId` | String | ID of the Category | "5eabf31ee0e9cd7b01a3a35e" |
