### Response Parameters

:::note

- Current type in <b>Status</b> will be :-

  - "EMPTY"<br/>
  - "WAITING_FOR_PAYMENT"<br/>
  - "WAIT_FOR_CONFIRMATION"<br/>
  - "CANCELLED"<br/>
  - "PROGRESSING"<br/>
  - "WAITING_FOR_COURIER"<br/>
  - "FINDING_COURIER"<br/>
  - "SHIPPING"<br/>
  - "COMPLETED"<br/>
  - "DELAYED"<br/>
  - "REJECTED"<br/>

- Current type in <b>State</b> will be :-

  - "ACCEPTED" , "REJECTED" , "CANCELED" , "REFUNDED"

- Current type in <b>Type</b> will be :-
